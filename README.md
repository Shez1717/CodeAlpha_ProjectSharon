# CodeAlpha_ProjectSharon
#Flash Cards App

An interesting flashcard application for a more intense learning and special memorization.It is quite special for University Students to tackle studying making it fun as possible.

##Features
-Study mode-Flip cards to reveal answers
-Shuffle mode -Card order for better studying navigating between previous and next.
-Responsive Design-It can work on desktop,mobile phone and even tablet
-The cards can be easily edited by the user anytime they want to which includes edit current,delete current and add flashcard.
